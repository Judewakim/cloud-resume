networking OSPF Network Routing Protocol [UNDER CONSTRUCTION]

>>> BLOGPOST 3: OSPF <<<

In this blogpost, I review Open Shortest Path First Cisco routing protocol. OSPF is a link-state routing protocol used in most switches and routers in modern networks.

Reference

Jermey's IT Lab YouTube Channel
Free CCNA | Configuring OSPF | CCNA 200-301 Complete Course

Quick Example

router ospf [process-id]
network [ip-address] [wildcard-mask] area [area-id]
show ip ospf [process-id]
